INTRODUCTION TO HTML

HTML is short for "Hypertext Markup Language"
It is the standard markup language for all webpages
It determines the layout of webpages. HTML was created by
Tim Berners Lee in the year 1960. The latest verion of HTML
is HTML 5. HTML follows a certain syntax distinct from
other programming languages. HTML can
be said to be a set of tags that conform to SGML. HTML consists of opening and closing tags respectively.
